## Core Guidelines

Before writing code, follow these principles:

- **Code Quality** - KISS, YAGNI, DRY, SOLID principles
- **Testing** - Test methodology and structure
- **Security** - OWASP Top 10, security checklist

## Quality Gates

All code must pass before commit:
- Linting passes
- Type checking passes (if applicable)
- Tests pass
- Build succeeds
